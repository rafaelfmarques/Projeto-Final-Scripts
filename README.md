# Ganhos e Gastos
Projeto desenvolvido cuja principal proposta é facilitar o controle de gastos e ganhos de um motoboy, diariamente, semanalmente, mensalmente ou até anualmente, mostrado tanto em uma tabela quanto graficamente.

O código foi desenvolvido usando:

<ul>
  <li>ReactJS</li>
  <li> Biblioteca Gráfica React-Apex-Chart</li>
</ul>
